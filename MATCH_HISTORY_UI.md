## Color Scheme

- **Victory**: Green (#4CAF50) for borders, headers, and positive values
- **Defeat**: Red (#F44336) for borders, headers, and negative values
- **MMR**: Gold (#FFD700) for MMR displays
- **Wins**: Green (#4CAF50)
- **Losses**: Red (#F44336)
- **Win Rate**: Blue (#2196F3)
- **Background**: Dark with transparency (rgba(0, 0, 0, 0.7))
- **Borders**: White with transparency (rgba(255, 255, 255, 0.2-0.4))

## Responsive Design

### Desktop (>600px)
- Cards: 400px wide
- Font sizes: 16-24px
- Full padding and spacing

### Mobile (<600px)
- Cards: 300px wide
- Font sizes: 14-20px
- Reduced padding and spacing
- Maintains full functionality

## Interaction States

### Hover Effects
- Match cards: Slight brightness increase
- Buttons: Background color change
- Arrows: Scale increase

### Click/Tap
- "View Replay" button: Launches replay in game engine
- Arrow buttons: Navigate carousel
- "Back to Menu" button: Returns to main menu

## Animation Details

### Card Transitions
- Position: Transform translate animation
- Scale: 0.85 to 1.0 (center card)
- Opacity: 0.5 to 1.0 (center card)
- Duration: 0.3 seconds
- Easing: ease function

### Screen Transitions
- Fade out current screen
- Fade in new screen
- Particle effects continue during transition

## Typography

- **Headers**: 'Doto' font, uppercase, letter-spacing
- **Body**: 'Doto' font, mixed case
- **Stats**: Bold weight for numbers
- **Labels**: Light weight, small caps

## Accessibility

- High contrast text
- Large clickable areas for buttons
- Clear visual feedback on interactions
- Keyboard navigation supported
- Screen reader friendly (semantic HTML)

## Data Display

### Match Duration Format
- Displayed as MM:SS (e.g., "12:34")
- Leading zero on seconds (e.g., "5:08")

### Timestamp Format
- Date: locale-specific (e.g., "2/9/2026")
- Time: 12-hour format with AM/PM

### MMR Change Format
- Positive: "+24" in green
- Negative: "-16" in red
- Zero: "0" in neutral color

### Faction Names
- Radiant: White/blue theme
- Aurum: Gold theme
- Velaris: Purple/dark theme
